PROBLEM:
--------
You see errors like:
  - "HTTP 404" in browser console
  - "Failed to load database"
  - Blank page

SOLUTION (3 SIMPLE STEPS):
---------------------------

STEP 1: Copy the database file
   
   Windows:
     Double-click: copy-database.bat
   
   Mac/Linux:
     Open Terminal and run:
       chmod +x copy-database.sh
       ./copy-database.sh
   
   Or use npm (all platforms):
     npm run prepare-db

STEP 2: Start the development server
   
   Open Terminal/Command Prompt and run:
     npm run dev

STEP 3: Open your browser
   
   Go to: http://localhost:5173

WHY IS THIS NECESSARY?
----------------------

The database file (complete-database.json) needs to be in the public/data/ 
folder for Vite to serve it. The copy script does this automatically.

You only need to do this ONCE. After that, Vite will copy it automatically 
every time you run "npm run dev" or "npm run build".
