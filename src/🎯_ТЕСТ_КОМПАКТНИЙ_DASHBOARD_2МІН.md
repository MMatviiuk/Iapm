### Крок 2: Перевірити Stat Cards (30 секунд)

**Що перевірити:**

✅ **4 картки статистики (Total, Today, Adherence, Remaining):**
- Картки НЕ займають весь екран
- Видно всі 4 картки без прокрутки
- Іконки у круглих контейнерах (не голі іконки)
- Числа читабельні але НЕ надто великі
- Відступи між картками невеликі

**Очікуваний вигляд:**
```
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│ [💊]     │ │ [📅]     │ │ [📈]     │ │ [⏰]     │
│ Total    │ │ Today    │ │ Adherenc │ │ Remainin │
│   10     │ │  3/10    │ │   95%    │ │    7     │
└──────────┘ └──────────┘ └──────────┘ └──────────┘
```

**НЕ має бути:**
```
┌─────────────────────────┐
│  💊  Total              │  ← Занадто велика
│                         │
│         10              │
│                         │
└─────────────────────────┘
```

### Крок 3: Перевірити Next Medication (20 секунд)

**Що перевірити:**

✅ **Картка "Next Medication":**
- Картка компактна (не займає половину екрану)
- Кнопка "Take Now" зелена та видима
- Кнопка НЕ надто велика (висота ~40-48px)
- Іконка Target у маленькому синьому кружечку
- Назва ліків та час видимі

**Очікуваний вигляд:**
```

│                                             │
```

**НЕ має бути:**
```
┌─────────────────────────────────────────────┐

│          in 2 hours                         │

│    │   Take Now ✅                   │     │  ← Кнопка надто велика

│    Aspirin 100mg                            │

│    Before meal                              │
│                                             │
└─────────────────────────────────────────────┘
```
