## 🚨 ЯКЩО НЕ БАЧИТЕ КНОПКИ

### Варіант 1: Кеш браузера (95% випадків)

**Chrome (Android):**
```
1. Меню (⋮) → Налаштування
2. Конфіденційність та безпека
3. Очистити дані перегляду
4. Позначити: Кеш + Cookies
5. Очистити
6. ЗАКРИТИ Chrome повністю (свайп вгору в списку додатків)
7. Відкрити знову
```

**Safari (iPhone):**
```
1. Налаштування iOS → Safari
2. Очистити історію та дані
3. ЗАКРИТИ Safari повністю (свайп вгору)
4. Відкрити знову
```

**АБО Режим Інкогніто:**
```
Chrome: ⋮ → Нова вкладка в режимі інкогніто
Safari: Іконка вкладок → Приватний перегляд
```

---

### Варіант 2: Розгорніть картку

Кнопки **Edit** та **Delete** показуються ТІЛЬКИ коли ви **розгорнули картку** dependent/patient.

**Як розгорнути:**
1. Натисніть на **будь-яку частину** картки dependent/patient
2. АБО натисніть на **стрілку вниз** (ChevronDown) праворуч
3. Картка розгорнеться і покаже медикаменти
4. Під кожним медикаментом з'являться кнопки

---

### Варіант 3: Немає медикаментів

Якщо у dependent/patient **немає медикаментів**, кнопки Edit/Delete не відображаються.

**Перевірте:**
- Anna Williams має 3 медикаменти (Aspirin, Vitamin D3, Omeprazole)
- Robert Williams має 2 медикаменти (Metformin, Lisinopril)
- John Smith має 4 медикаменти (Lisinopril, Metformin, Atorvastatin, Aspirin)

1. **Print All (вгорі)**
