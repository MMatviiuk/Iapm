## 👤 РОЛЬ 1: PATIENT (Myself) - Пенсионер

### 🎯 Главная Цель:
**"Я хочу знать ЧТО и КОГДА принимать СЕГОДНЯ"**

### 📱 Путь #1: Первый Запуск
```
1. Landing Page → Sign Up
2. Выбор роли: "For Myself" (синяя карточка)
3. Заполнение формы: Email, Password, Name, DOB
4. Onboarding (4 слайда) → Dashboard
5. Empty State: [+ Add Your First Medication]
6. Wizard (3 шага): Essential → When → Optional
7. Success! → Today's Schedule
```

### 📱 Путь #2: Ежедневное Использование
```
1. Открывает приложение
2. Auto Login (Remember Me) ← NEW!
3. Видит: "Next Medication: Lisinopril 8:00 AM [Take Now]"
4. Нажимает [Take Now]
5. ✓ Зеленая галочка + Toast "Great Job! 💊"
6. Progress bar: 1/1 (100%)
```

### 📊 Навигация (Sidebar)
```
┌─ Overview (collapsible) ───────────┐
│  📊 Dashboard                      │
│  📅 Today                          │
│  📆 Week View                      │
├─ Tracking (collapsible) ───────────┤
│  🕐 History                        │
│  💊 Medications                    │
│  🔔 Notifications                  │
├─ Personal (collapsible) ───────────┤
│  🏆 Achievements                   │
│  ⚙️ Settings                       │
└─────────────────────────────────────┘
[+ Add Medication] ← BLUE BUTTON
```

**По умолчанию:** Только "Overview" открыт

### 🎚️ 3 Режима для Patient

#### 1. Full Dashboard (Default)
- Все виджеты видны
- Collapsible секции можно раскрывать
- Для тех, кто хочет видеть статистику

#### 2. Focused Dashboard (Settings toggle)
- Next Medication + Today's Progress
- Collapsible секции закрыты by default
- Меньше информации = меньше путаницы

#### 3. Simplified Mode (Settings toggle)
- Показывает ТОЛЬКО Today's Schedule
- Скрывает Dashboard, Week View, History
- Sidebar: только Today, Settings
- Для самых пожилых (80+)

### 🎯 Критичные Требования
- ✅ Кнопки: 56-64px (большие для пенсионеров)
- ✅ Шрифт: 18-20px (увеличенный)
- ✅ Иконки: 28-32px (крупные)
- ✅ Контраст: WCAG AAA (7:1)
- ✅ Только ОДИН вид (компактный УДАЛЕН)

## 👨‍⚕️ РОЛЬ 3: DOCTOR (Медицинский Работник)

**"Я хочу видеть, СОБЛЮДАЮТ ли мои пациенты назначения"**

### 📱 Путь #1: Первый Запуск
```
1. Landing Page → Sign Up
2. Выбор роли: "Healthcare Professional" (фиолетовая карточка)
3. Заполнение формы: Email, Password, Name, Specialty, License
4. Onboarding (4 слайда) → Patients Dashboard
5. Empty State: [+ Invite Your First Patient]
6. Форма: Patient Email, Name (optional), Message

```

### 📱 Путь #2: Ежедневное Использование
```
1. Открывает приложение
2. Auto Login
```

```
1. Patients Dashboard → John Smith

   └─ Optional

5. Toast: "✅ Lisinopril prescribed to John Smith!"
6. Пациент получает notification
```

```
(Без collapsible - только 4 пункта)
```

- ✅ Список пациентов (фото, имена, статистика)
- ✅ Назначение лекарств (Prescribe Medication)

- ✅ Cohort Analytics (все пациенты)
- ✅ At-risk patients (adherence <80%)
