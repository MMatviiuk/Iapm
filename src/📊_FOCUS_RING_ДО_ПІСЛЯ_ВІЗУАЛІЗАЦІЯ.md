## 🔬 ТЕХНІЧНІ ЗМІНИ

### Зміна 1: Додано padding до форми
```tsx
// ДО
<form onSubmit={handleSubmit} className="space-y-6">
  {/* focus ring обрізається */}
</form>

// ПІСЛЯ
<form onSubmit={handleSubmit} className="space-y-6 px-1">
  {/* px-1 = 4px padding з кожного боку для 3px ring */}
</form>
```

**Простір:**
```
ДО:  |RING|||INPUT||||||||||||||||||||||||RING|  ← обрізано
ПІСЛЯ: | |RING|||INPUT|||||||||||||||||||RING| |  ← видно
       ↑ 4px                              4px ↑
```

### Зміна 2: Збільшено відступ іконки
```tsx
// ДО
<button className="absolute right-4 top-1/2 -translate-y-1/2">
  {/* right-4 = 16px - занадто близько */}
  <Eye className="w-5 h-5" />
</button>

// ПІСЛЯ
<button className="absolute right-5 top-1/2 -translate-y-1/2">
  {/* right-5 = 20px - правильно */}
  <Eye className="w-5 h-5" />
</button>
```

**Позиція:**
```
ДО:  |||INPUT|||||||||||||||||| 👁️|  ← поза рамкою
ПІСЛЯ: |||INPUT||||||||||||||||  👁️ |  ← всередині
                            20px ↑
```
