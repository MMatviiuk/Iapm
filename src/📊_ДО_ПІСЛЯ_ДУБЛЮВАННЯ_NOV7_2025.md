## 💡 ЧОМ�� ЦЕ КРАЩЕ

### 1. Чіткі ролі екранів
- **Dashboard** = Подивитися статистику
- **Today** = Позначити ліки як прийняті

### 2. Без дублювання
- Кожен список ліків показується тільки РАЗ
- Не плутає користувача

### 3. Літні користувачі
- Один основний екран (Today)
- Великі круглі кнопки (56-60px)
- Просто зрозуміти: "Today = мої ліки"

### 4. Менше когнітивного навантаження
- Не треба вибирати між 3 екранами
- Dashboard → погляд, Today → дії
