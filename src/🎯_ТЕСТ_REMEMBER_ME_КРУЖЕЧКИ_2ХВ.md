## ❌ ЯК ЗНАЙТИ ПРОБЛЕМИ

### Проблема 1: Remember Me не працює

**Симптоми:**
- Після оновлення сторінки → logout
- localStorage не містить `authTokenExpiry`
- Консоль НЕ показує "Token expiry saved"

**Рішення:**
```bash
# Перевірити App.tsx handleLogin:
if (data.expiresAt) {
  localStorage.setItem('authTokenExpiry', data.expiresAt.toString());
}
# Має бути присутнім!
```

---

### Проблема 2: Кружечки всі білі

**Симптоми:**
- Всі кола білі/сірі
- Немає червоних/жовтих/зелених

**Рішення:**
```bash
# Перевірити MainSchedule.tsx checkbox:
className={`... ${
  med.mealTiming === 'before meal' ? 'bg-red-500' :
  med.mealTiming === 'with meal' ? 'bg-yellow-400' :
  med.mealTiming === 'after meal' ? 'bg-green-500' :
  'bg-white'
}`}
# Має бути присутнім!
```
