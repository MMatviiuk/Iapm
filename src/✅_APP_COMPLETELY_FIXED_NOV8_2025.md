### 1. HARD REFRESH (обов'язково!)
**Windows/Linux:**
```
Ctrl + Shift + R
```

**Mac:**
```
Cmd + Shift + R
```

### 2. Якщо не допомогло - Очистіть кеш повністю:

**Chrome/Edge:**
1. F12 (відкрити DevTools)
2. Правий клік на кнопку Reload (⟳)
3. Оберіть "Empty Cache and Hard Reload"

**Firefox:**
1. Ctrl + Shift + Delete
2. Оберіть "Cache"
3. "Clear Now"

### 3. Останній варіант - Інкогніто:
```
Ctrl + Shift + N (Windows)
Cmd + Shift + N (Mac)
```
