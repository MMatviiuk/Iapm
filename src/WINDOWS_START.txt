TROUBLESHOOTING:
-----------------

Problem: "npm: command not found"
Solution: Install Node.js from https://nodejs.org/

Problem: "HTTP 404" error in browser
Solution: Run copy-database.bat again, then npm run dev

Problem: "Port 5173 already in use"
Solution: 
  netstat -ano | findstr :5173
  taskkill /PID <PID> /F
  npm run dev

Problem: Build fails
Solution:
  rmdir /s /q dist node_modules\.vite
  npm run dev
