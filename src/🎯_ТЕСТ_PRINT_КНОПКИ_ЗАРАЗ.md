## 🧪 Швидкий Тест (5 хвилин)

### 1. Запусти застосунок
```bash
npm run dev
```

---

### 2. ТЕСТ ПАЦІЄНТА

**Увійди як пацієнт:**
```
Email: patient@demo.com
Password: demo123
```

**Перевір Print кнопку:**
1. Перейди на **Today** (або MainSchedule)
2. **Шукай кнопку Print** - вгорі праворуч, поруч з Moon/Sun (темна тема)
3. **Має бути:**
   - ✅ Синя кругла кнопка (56-60px)
   - ✅ Іконка принтера 🖨️
   - ✅ Поруч з Moon (🌙) або Sun (☀️)

**Клікни Print:**
1. Клікни на 🖨️ кнопку
2. **Очікування:**
   - ✅ Переходить на сторінку "Print Schedule"
   - ✅ Показує ім'я пацієнта вгорі
   - ✅ Показує тиждень (наприклад, "Nov 3 - Nov 9")
   - ✅ Таблиця з днями тижня (Mon-Sun)
   - ✅ Всі медикаменти в клітинках
   - ✅ Кнопка "Print" вгорі

**Друк:**
1. Клікни "Print" (вгорі на сторінці Print Schedule)
2. **Очікування:**
   - ✅ Відкривається діалог друку браузера
   - ✅ Попередній перегляд показує таблицю
   - ✅ Орієнтація: Landscape (альбомна)

---

### 3. ТЕСТ ОПІКУНА

**Вийди та увійди як опікун:**
```
Email: caregiver@demo.com
Password: demo123
```

**Перевір кнопки Print:**
1. На dashboard бачиш список опікуваних
2. **Розкрий будь-якого опікуваного** (клікни на картку)
3. **Має з'явитися:**
   - ✅ Список медикаментів
   - ✅ Кожен медикамент має кнопки: Print / Edit / Delete
   - ✅ Внизу Quick Actions: **Print Week** + Add Medication

**Клікни Print (біля медикаменту):**
1. Клікни "Print" біля будь-якого медикаменту
2. **Очікування:**
   - ✅ Переходить на Print Schedule
   - ✅ Показує ім'я опікуваного (наприклад, "Margaret Williams")
   - ✅ Показує ім'я опікуна ("Catherine Bennett - Caregiver")
   - ✅ **ВСІ медикаменти** опікуваного в таблиці (не тільки один!)

**Клікни Print Week (Quick Actions):**
1. Клікни "Print Week" (велика кнопка внизу)
2. **Очікування:**
   - ✅ Те саме - переходить на Print Schedule
   - ✅ Всі медикаменти опікуваного
   - ✅ Готово для друку

## ⚠️ Можливі Проблеми

### 1. Кнопка Print не видима:
```bash
# Перезавантаж сторінку
Ctrl+Shift+R (Windows/Linux)
Cmd+Shift+R (Mac)
```

### 2. Print відкриває popup замість PrintSchedule:
- Має бути виправлено! Тепер відкриває PrintSchedule
- Якщо все ще popup - очисти кеш браузера

### 3. PrintSchedule порожній:
- Перевір чи є медикаменти у пацієнта/опікуваного
- Додай медикаменти через "Add Medication"
