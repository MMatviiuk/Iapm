## 🔧 Якщо FAIL

### 1. Очистіть кеш:

**Chrome (Android):**
```
⋮ → Налаштування → Конфіденційність
→ Очистити дані перегляду
→ Кеш + Cookies → Очистити
```

**Safari (iPhone):**
```
Налаштування iOS → Safari
→ Очистити історію та дані
```

### 2. Режим інкогніто:

**Chrome:**
```
⋮ → Нова вкладка в режимі інкогніто
```

**Safari:**
```
Іконка вкладок → Приватний перегляд → +
```

### 3. Перезавантажте телефон:
```
Вимкніть → Зачекайте 10 сек → Увімкніть
```
