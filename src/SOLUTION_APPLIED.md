## Why This Happens

The database file needs to be in TWO locations:

### Source (for developers):
```
/data/complete-database.json
```
- Tracked by git
- Source of truth
- Used for development

### Public (for browser):
```
/public/data/complete-database.json
```
- Served by Vite
- Accessible by browser
- Generated by copy script

When you request `/data/complete-database.json` in the browser, Vite serves it from `/public/data/complete-database.json`.
